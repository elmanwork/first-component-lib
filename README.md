# first-component-lib

> Made with create-react-library

[![NPM](https://img.shields.io/npm/v/first-component-lib.svg)](https://www.npmjs.com/package/first-component-lib) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save first-component-lib
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'first-component-lib'
import 'first-component-lib/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
